# Gym-Congestion-Analysis
Beginner SQL project analyzing gym congestion patterns across different times of the day.
Gym Congestion Analysis

This project analyzes when the gym is most and least busy.

Over a two-week period I collected observational data during three time blocks:
- Morning
- Noon
- Evening

I stored the data in a SQL table and used queries such as
GROUP BY and WHERE to analyze patterns.

Key findings:
- Evening had the most congestion.
- Weekdays were generally busier.
- Noon was the least busy time.

Tools used:
SQL, Excel
